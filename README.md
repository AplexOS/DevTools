## 1. Branch Namimg Specification

* **branch namimg format**: (product name)-(tool name)-(system type)
  * **product name**:
    * SBC-7112S;
    * SBC-7112S-Q;
  * **system type**:
    * Linux3.0.35;
    * Linux3.0.35-Qt4.8.5;
* **E.g**  
`SBC-7112S`(product name)`-burning`(tool name)`-Linux3.0.35`(system type) = `SBC-7112S-burning-Linux3.0.35`
